
CELLARCOUNT 2026 – ZIP 9 (BYG OVENPÅ ZIP 8)

Denne ZIP er KUN en udvidelse.
Intet eksisterende overskrives.

GØR SÅDAN:
1. Brug din fungerende ZIP 8 som base
2. Opret mappen /js hvis den ikke findes
3. Læg count.js i /js
4. Tilføj dette script nederst i index.html (før </body>):

<script src="js/count.js"></script>

5. Tilføj + / − knapper i tabellen:
<button onclick="incrementWine(vin.vinId)">+</button>
<button onclick="decrementWine(vin.vinId)">−</button>

DETTE ZIP FJERNER INTET.
